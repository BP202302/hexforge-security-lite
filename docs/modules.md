# Modules

HexForge Lite includes 16 passive modules:

1. Security headers
2. Clickjacking
3. CORS policy
4. Cookie flags
5. Cache policy
6. Redirect policy
7. Content type
8. Metadata exposure
9. HTML comments
10. Email/token exposure
11. External resources
12. Mixed content
13. Forms basics
14. Robots/sitemap
15. Client surface mapping
16. TLS basics

Each module returns structured findings. Validators decide final severity, confidence and category.
